Basic Web Concepts


Summary

The world wide web is a collection of html docs and other types of files. HTML is the main document typee and glues everything together. It is made up of text, markup, references to other docs and links to other pages.

HTML Tags

  • HTML tags <> and (has opening and closing tags)
  • xamples: bold (b), italics (em)
  • HTML attributes have names and values.
  • Image tags - needs a source and alt (not required but useful for when images dont render and for people that rely on readers)
  • Whitespace (br tags and p tags)
  • inline (<br>, <em> and <img>,<span>) vs block (<p>, <div>)
    • *Block inserts formatting while in line does not
